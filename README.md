# BitLocker 密码恢复工具

## 📋 工具概述

这是一个专为 Windows 系统设计的 BitLocker 密码恢复工具，通过多线程并行尝试密码组合的方式，帮助用户找回 BitLocker 加密驱动器的密码。

### 核心功能
- 🎯 多线程并行密码尝试
- 🔍 自动检查驱动器状态和 BitLocker 保护类型
- 🛠️ 支持自定义关键字生成密码组合
- 🔤 自动生成关键字的大小写变体
- ⚡ 快速测试关键密码
- 📊 实时显示尝试进度和性能指标
- 🔒 PowerShell 解锁引擎
- ⭐ 支持 8 种特殊符号
- 🔐 12 种密码拼接方式

## 🚀 快速开始

### 1. 系统要求

## 🎮 使用流程

### 基本使用步骤
1. **启动程序**：以管理员身份运行脚本
2. **输入驱动器盘符**：指定要解锁的驱动器（如 E）
3. **状态检查**：程序自动验证驱动器状态和 BitLocker 保护类型
4. **输入关键字**：输入密码相关的关键字（如姓名、用户名等）
5. **快速测试**：程序先测试一些关键密码组合
6. **生成密码**：基于关键字生成密码组合
7. **设置线程数**：输入要使用的线程数量（建议 4-8）
8. **开始尝试**：多线程并行尝试密码
9. **结果反馈**：成功找到密码或报告失败原因

## ⚙️ 技术细节

### 密码生成逻辑

| 组件 | 描述 | 示例 |
|------|------|------|
| str1 | 用户自定义关键字 | "zhang", "san" |
| str2 | 年份和数字组合 | "2000"-"2024", "00"-"24" |
| str3 | 特殊日期 | "1229", "0223", "0406" |
| str4 | 特殊符号 | "", "!", "@", "#", "$", "%", "^", "&" |

**生成公式：**
1. 关键字 + 特殊符号 + 年份（如 "zhang!2024"）
2. 关键字 + 特殊符号 + 日期（如 "zhang@0406"）
3. 关键字 + 特殊符号 + 年份 + 日期（如 "zhang#20240406"）
4. 关键字 + 年份 + 日期（如 "zhang20240406"）
5. 关键字 + 年份（如 "zhang2024"）
6. 关键字 + 日期（如 "zhang0406"）
7. 年份 + 日期（如 "20240406"）
8. 日期（如 "0406"）
9. 关键字 + 年份 + 日期 + 特殊符号（如 "zhang20240406!"）
10. 关键字 + 年份 + 特殊符号（如 "zhang2024@"）
11. 关键字 + 日期 + 特殊符号（如 "zhang0406#"）
12. 年份 + 日期 + 特殊符号（如 "20240406$"）

### 线程配置建议

| CPU 核心数 | 推荐线程数 | 备注 |
|-----------|-----------|------|
| 2-4 核心 | 4-8 线程 | 平衡性能与稳定性 |
| 4-8 核心 | 8-16 线程 | 最大化利用多核心 |
| 8+ 核心 | 16-32 线程 | 高性能配置 |

## 🔧 故障排除

### 常见问题

| 问题 | 可能原因 | 解决方案 |
|------|---------|----------|
| "需要管理员权限" | 未以管理员身份运行 | 右键 → "以管理员身份运行" |
| "无法访问驱动器" | 驱动器未连接或损坏 | 检查物理连接和驱动器状态 |
| "未找到密码保护器" | 驱动器使用其他保护方式 | 检查 manage-bde -protectors 输出 |
| "PowerShell 错误" | PowerShell 模块未启用 | 启用 BitLocker PowerShell 模块 |
| 编码错误 | 系统区域设置不匹配 | 使用 gbk 编码运行 |
